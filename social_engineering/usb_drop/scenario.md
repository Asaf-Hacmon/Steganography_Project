# USB Drop Attack — תרחיש הפלת USB

## תיאור ההתקפה

התוקף מניח USB Drive במקום ציבורי (חדר קפה, חניון, לובי של חברה, ספרייה).
ה-USB מעוצב כך שנראה שנשכח על ידי מישהו — עם תווית מפתה.

---

## תוכן ה-USB

```
USB Drive/
├── autorun.inf          ← מריץ את ה-EXE אוטומטית בחיבור (Windows XP–7)
├── ImageViewer.exe      ← הטרויאני שלנו
├── stego_image.png      ← תמונה עם פאילוד מוטמע
└── תמונות מהטיול 2026/  ← תיקייה עם תמונות תמימות להסוואה
    ├── IMG_001.jpg
    ├── IMG_002.jpg
    └── ...
```

---

## תווית ה-USB (הנדסה חברתית)

מודפסת על מדבקה ומודבקת על ה-USB:

```
┌─────────────────────────────┐
│  📸 תמונות אילת 2026        │
│     דניאל כהן               │
│  ☎ 050-1234567              │
└─────────────────────────────┘
```

**למה זה עובד:** מוצא ה-USB מרגיש שהוא עושה טובה בהחזרת הנכס לבעלים — ומתחבר כדי לבדוק למי להתקשר. הסקרנות ("מה יש בתמונות?") מניעה פתיחת הקבצים.

---

## וקטורי הפעלה

### וקטור 1 — Autorun (Windows XP / 7 ישן)
חיבור ה-USB מריץ את `ImageViewer.exe` **אוטומטית** בלי שהמשתמש לוחץ על כלום.

```
[autorun]
open=ImageViewer.exe
action=פתח אלבום תמונות
label=אלבום תמונות 📸
```

> **הערה:** ב-Windows 8+ Autorun מושבת כברירת מחדל לקבצי EXE.
> עדיין פופולרי ברשתות ארגוניות עם מחשבים ישנים.

### וקטור 2 — הפעלה ידנית (Windows 10/11)
המשתמש פותח את ה-USB ורואה:
- תיקייה: `תמונות מהטיול 2026` — נראה תמים
- קובץ: `ImageViewer.exe` עם אייקון תמונה 🖼

הוא לוחץ לפתוח את ה-EXE בחשיבה שזה "תוכנת הצפייה בתמונות שב-USB".

### וקטור 3 — LNK Shortcut (מתקדם)
במקום EXE ישיר, יצירת קובץ `.lnk` (קיצור דרך) שנראה כמו תמונה:
```
תמונות הטיול.lnk  →  מריץ: ImageViewer.exe /hide
```
אייקון מוגדר כ-JPG — המשתמש לא שם לב שזה קיצור דרך.

---

## מה קורה לאחר ההפעלה

1. `ImageViewer.exe` נפתח ומציג את `stego_image.png` — נראה תמים לחלוטין
2. ברקע: מחלץ את הפאילוד מה-LSB של התמונה
3. מריץ את הקוד הנסתר:
   - יוצר `pwned.txt` על שולחן העבודה
   - מציג חלון התראה

---

## הגנות נגד USB Drop

| הגנה | תיאור |
|------|--------|
| מדיניות ארגונית | חסימת USB לא מורשים דרך Group Policy |
| הדרכת משתמשים | לא לחבר USB שנמצא בשטח |
| Endpoint Security | סריקת קבצים אוטומטית בחיבור USB |
| AppLocker | מניעת הרצת EXE מדיסקים חיצוניים |

---

## למה זה עובד — פסיכולוגיה

- **סקרנות:** "מה יש ב-USB הזה?"
- **נדיבות:** "אחזיר אותו לבעלים, אבדוק קודם למי"
- **אמון:** נראה אישי ולא מסחרי — לא ספאם
- **דחיפות נמוכה:** אין לחץ זמן → פחות חשד
